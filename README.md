# milinks
https://milinks.info
bookmarks manager with tags support

INSTALL :
git clone  
cd milinks  
virtualenv venv -p python3  
source /venv/bin/activate  
pip install -r requirements.txt  
cd project    
bower install
grunt ?  
cd ..
python manage.py create_db
python manage.py runserver
